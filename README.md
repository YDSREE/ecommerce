# E-Commerce Web Application

A fully responsive E-commerce web application built using React.js that allows users to browse products, add items to the cart, manage quantities, and place orders. This project demonstrates modern frontend development practices including component-based architecture, state management, API integration, and responsive UI design.

 **Live Demo:** https://ecommerce-six-drab-45.vercel.app

---

##  Features

-  User Authentication (Login & Register)
-  Product Listing
-  Add to Cart & Remove from Cart
-  Increase / Decrease Product Quantity
-  Fast UI Rendering
-  Fully Responsive Design
-  Backend API Integration

---
